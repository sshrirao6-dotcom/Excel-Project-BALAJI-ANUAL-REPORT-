# Excel-Project-BALAJI-ANUAL-REPORT-
“This project analyzes the annual sales performance of Balaji Store using Excel, focusing on customer trends, order patterns, and revenue insights. An interactive dashboard was created using Pivot Tables and Charts to support quick business decision-making.”


1. Project Title
Balaji Store – Annual Sales Analysis Report (Excel Dashboard Project)

2. Short Description / Purpose
Analyzed one-year store performance using Excel to understand sales trends, order patterns, customer behaviour, and revenue insights.
Designed an interactive dashboard for quick decision-making by management.

3. Tools / Tech Stack
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
Conditional Formatting
Data Cleaning & Transformation

4. Data Used / Data Storage
Monthly sales and orders dataset including:
Customer demographics
Product categories
Order status
Sales channel data
State-wise performance

5. Key Features / Highlights
Created a dynamic dashboard showing monthly sales performance.
Visualized Sales vs Orders to study seasonal peaks.
Compared Male vs Female customer purchase patterns.
Built State-wise sales analysis to identify top-performing regions.
Designed charts for Order Status, Order Channels, and Average Order Value by Gender.
Cleaned and transformed raw data for accurate reporting.

6. Example Problem Solved
“Which state generated the highest sales and through which channel?”
“Which gender contributes more to total sales?”
“How are monthly order trends changing throughout the year?”
